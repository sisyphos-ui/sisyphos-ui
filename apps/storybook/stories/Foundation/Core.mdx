# Core Package

`@sisyphos-ui/core` paketi, Sisyphos UI design system'in temelini oluşturur. Tema sistemi ve design tokens bu pakette bulunur.

## 📦 Paket İçeriği

### 1. Theme System
Runtime tema customizasyonu için fonksiyonlar:

- `applyTheme()` - Tema uygula
- `setThemeMode()` - Light/Dark mode
- `createTheme()` - Type-safe tema oluştur
- `mergeThemes()` - Temaları birleştir

### 2. Design Tokens

#### CSS Variables
Tüm tasarım değerleri CSS Variables olarak tanımlanmıştır:

```css
:root {
  --sisyphos-color-primary: #ff7022;
  --sisyphos-spacing-md: 16px;
  --sisyphos-radius-s: 8px;
  --sisyphos-font-size-base: 16px;
}
```

#### SCSS Variables
SCSS kullanıcıları için SCSS variables:

```scss
@use "@sisyphos-ui/core/tokens/variables" as *;

.my-component {
  padding: $spacing-md;      // → var(--sisyphos-spacing-md, 16px)
  border-radius: $radius-s;  // → var(--sisyphos-radius-s, 8px)
  font-size: $base-size;     // → var(--sisyphos-font-size-base, 16px)
}
```

### 3. Mixins

Reusable SCSS mixins:

```scss
@use "@sisyphos-ui/core/tokens/mixins" as *;

// Button variants
@include button-variant($variant, $color, $bg-color);

// Responsive breakpoints
@include mobile { }
@include tablet { }
@include desktop { }
```

## 🎯 Kullanım

### Installation

```bash
npm install @sisyphos-ui/core
# veya
pnpm add @sisyphos-ui/core

yarn add @sisyphos-ui/core
```


```tsx
import { applyTheme, setThemeMode } from '@sisyphos-ui/core';

// Tema uygula
```

```scss
.my-component {
  padding: $spacing-md;
  color: var(--sisyphos-color-primary);
}
```

## 🎯 SCSS Implementation Examples

### Example 1: Custom Component (Using SCSS Tokens)

**Senaryo:** Kendi component'inizi oluşturuyorsunuz ve design tokens kullanmak istiyorsunuz.

**File Structure:**

```
src/
├── components/
│   ├── MyButton/
│   │   ├── MyButton.tsx
│   │   └── MyButton.scss
```

**Implementation:**

```scss
// src/components/MyButton/MyButton.scss
/* ✅ CORRECT - Use SCSS tokens */
@use "@sisyphos-ui/core/tokens/variables" as *;
@use "@sisyphos-ui/core/tokens/mixins" as *;

.my-button {
  /* Spacing - Use SCSS variables */
  padding: $spacing-s $spacing-md;
  margin: $spacing-xs;

  /* Typography - Use SCSS variables */
  font-size: $base-size;
  font-weight: $font-weight-medium;
  line-height: $line-height;
  font-family: $font-family;

  /* Colors - Use CSS variables (theme-aware) */
  color: var(--sisyphos-color-neutral-dark, #212b36);
  background: var(--sisyphos-color-primary, #ff7022);
  border-color: var(--sisyphos-color-primary-dark, #007867);

  /* Border Radius - Use SCSS variables */
  border-radius: $radius-md;
  border: 1px solid transparent;

  /* Transitions - Use SCSS variables */
  transition: all $duration-s;

  /* Z-Index - Use SCSS variables */
  z-index: $z-index-overlay;

  /* Hover state */
  &:hover {
    background: var(--sisyphos-color-primary-light, #ff8637);
  }

  /* Focus state */
  &:focus-visible {
    outline: 2px solid var(--sisyphos-color-primary, #ff7022);
    outline-offset: 2px;
  }

  /* Disabled state */
  &:disabled {
    opacity: $opacity-xs;
    cursor: not-allowed;
    pointer-events: none;
  }
}

/* ❌ WRONG - Hard-coded values */
.my-button-wrong {
  padding: 16px 24px;        /* ❌ Use $spacing-md $spacing-lg */
  color: #212b36;            /* ❌ Use var(--sisyphos-color-neutral-dark) */
  background: #ff7022;       /* ❌ Use var(--sisyphos-color-primary) */
  border-radius: 12px;       /* ❌ Use $radius-md */
  font-size: 16px;           /* ❌ Use $base-size */
  z-index: 1000;             /* ❌ Use $z-index-overlay */
}
```

**TypeScript Component:**

```tsx
// src/components/MyButton/MyButton.tsx
import React from 'react';
import './MyButton.scss';

interface MyButtonProps extends React.ButtonHTMLAttributes<HTMLButtonElement> {
  variant?: 'primary' | 'secondary';
  size?: 'small' | 'medium' | 'large';
}

export const MyButton: React.FC<MyButtonProps> = ({
  variant = 'primary',
  size = 'medium',
  className = '',
  children,
  ...props
}) => {
  const classes = [
    'my-button',
    `my-button--${variant}`,
    `my-button--${size}`,
    className,
  ]
    .filter(Boolean)
    .join(' ');

  return (
    <button className={classes} {...props}>
      {children}
    </button>
  );
};
```

**Best Practice:**
- ✅ SCSS variables kullan (spacing, typography, etc.)
- ✅ CSS variables kullan (colors - theme-aware)
- ✅ Hard-coded değerler YOK
- ✅ Simplified BEM naming convention

### Example 2: Using SCSS Mixins

**Senaryo:** Reusable button variant mixin kullanmak istiyorsunuz.

**Implementation:**

```scss
// src/components/CustomButton/CustomButton.scss
@use "@sisyphos-ui/core/tokens/variables" as *;
@use "@sisyphos-ui/core/tokens/mixins" as *;

.custom-button {
  /* Base styles */
  padding: $spacing-s $spacing-md;
  border-radius: $radius-md;
  font-size: $base-size;
  font-weight: $font-weight-medium;
  cursor: pointer;
  transition: all $duration-s;

  /* Use mixin for variants */
  @include button-variant(
    "contained.primary",
    var(--sisyphos-color-neutral-lighter, #fff),
    var(--sisyphos-color-primary-contained, #ff7022),
    var(--sisyphos-color-primary-contained, #ff7022),
    $radius-md
  );

  /* Custom variants */
  &.custom-primary {
    background: var(--sisyphos-color-primary, #ff7022);
    color: var(--sisyphos-color-neutral-lighter, #fff);

    &:hover {
      background: var(--sisyphos-color-primary-light, #ff8637);
    }
  }

  &.custom-secondary {
    background: transparent;
    color: var(--sisyphos-color-primary, #ff7022);
    border: 1px solid var(--sisyphos-color-primary, #ff7022);

    &:hover {
      background: var(--sisyphos-color-primary-lighter, #c8fad6);
    }
  }
}
```

**Available Mixins:**

```scss
// Responsive breakpoints
@include mobile {
  /* Mobile styles */
  font-size: $size-sm;
}

@include tablet {
  /* Tablet styles */
  font-size: $base-size;
}

@include desktop {
  /* Desktop styles */
  font-size: $size-lg;
}

// Box styling
@include box-style(
  $border-color: var(--sisyphos-color-neutral-darker, #919eab33),
  $border-radius: $radius-md,
  $box-shadow: 0px 4px 12px rgba(0, 0, 0, 0.1),
  $bg-color: var(--sisyphos-color-neutral-lighter, #fff)
);
```

**Senaryo:** CSS Variables'ı direkt kullanmak istiyorsunuz (SCSS olmadan).

**Implementation:**

```css
/* src/styles/my-component.css */
.my-component {

  border-color: var(--sisyphos-color-neutral-darker, #919eab33);

  /* Spacing - Use CSS variables (with fallback) */

  margin: var(--sisyphos-spacing-s, 10px);

  /* Typography - Use CSS variables */
  font-size: var(--sisyphos-font-size-base, 16px);
  font-weight: var(--sisyphos-font-weight-medium, 500);
  font-family: var(--sisyphos-font-family, "Manrope", sans-serif);

  /* Border Radius - Use CSS variables */
  border-radius: var(--sisyphos-radius-md, 12px);

  /* Opacity - Use CSS variables */
  opacity: var(--sisyphos-opacity-xs, 0.6);

  /* Duration - Use CSS variables */
  transition: all var(--sisyphos-duration-s, 250ms);

  /* Z-Index - Use CSS variables */
  z-index: var(--sisyphos-z-index-overlay, 900);
}
```

**TypeScript Component (CSS):**

```tsx

```

**Best Practice:**
- ✅ CSS variables direkt kullanılabilir (SCSS gerekmez)
- ✅ Fallback değerleri ekle (`var(--sisyphos-color-primary, #ff7022)`)

```typescript
// Theme functions
export function applyTheme(config: ThemeConfig): void {
  const root = document.documentElement;
  
  // Colors
  if (config.colors) {
    if (typeof config.colors.primary === 'string') {
      root.style.setProperty('--sisyphos-color-primary', config.colors.primary);
    }
    // ... other colors
  }
  
  // Spacing
  if (config.spacing) {
    Object.entries(config.spacing).forEach(([key, value]) => {
      root.style.setProperty(`--sisyphos-spacing-${key}`, `${value}px`);
    });
  }
  
  // ... other tokens
}

export function setThemeMode(mode: 'light' | 'dark'): void {
  const root = document.documentElement;
  root.classList.remove('sisyphos-theme-light', 'sisyphos-theme-dark');
  root.classList.add(`sisyphos-theme-${mode}`);
}
```

## 🔗 Integration Patterns

### Pattern 1: Next.js Integration

```tsx
// app/layout.tsx (Next.js App Router)
import '@sisyphos-ui/core/styles.scss';
import { applyTheme } from '@sisyphos-ui/core';

// Apply theme at layout level
applyTheme({
  colors: { primary: '#007bff' },
});

export default function RootLayout({ children }) {
  return (
    <html lang="en">
      <body>{children}</body>
    </html>
  );
}
```

### Pattern 2: Vite + React Integration

```tsx
// src/main.tsx
import React from 'react';
import ReactDOM from 'react-dom/client';
import '@sisyphos-ui/core/styles.scss';
import { applyTheme } from '@sisyphos-ui/core';
import App from './App';

// Apply theme before rendering
applyTheme({
  colors: { primary: '#007bff' },
});

ReactDOM.createRoot(document.getElementById('root')!).render(
  <React.StrictMode>
    <App />
  </React.StrictMode>
);
```

### Pattern 3: Create React App Integration

```tsx
// src/index.tsx
import React from 'react';
import ReactDOM from 'react-dom/client';
import '@sisyphos-ui/core/styles.scss';
import { applyTheme } from '@sisyphos-ui/core';
import './index.css';
import App from './App';

applyTheme({
  colors: { primary: '#007bff' },
});

const root = ReactDOM.createRoot(
  document.getElementById('root') as HTMLElement
);

root.render(
  <React.StrictMode>
    <App />
  </React.StrictMode>
);
```

## 🐛 Troubleshooting

### Problem 1: SCSS Variables Çalışmıyor

**Çözüm:**

```scss
/* ✅ Doğru: @use ile import et */
@use "@sisyphos-ui/core/tokens/variables" as *;

.my-component {
  padding: $spacing-md; /* ✅ Çalışır */
}

/* ❌ Yanlış: @import kullanma (deprecated) */
@import "@sisyphos-ui/core/tokens/variables"; /* ❌ Çalışmaz */
```

### Problem 2: CSS Variables Çalışmıyor

**Çözüm:**

```tsx
// ✅ Styles'i import ettiniz mi?
import '@sisyphos-ui/core/styles.scss'; // En önce!

// ✅ CSS variables tanımlı mı? (default-theme.scss yüklendi mi?)

```

```scss
/* ✅ Doğru: @use ile import et */
@use "@sisyphos-ui/core/tokens/mixins" as *;
```
